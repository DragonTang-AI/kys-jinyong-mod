# KYS-CPP 数值系统重构设计文档

## 修改日期
2026-05-19

## 概述
本次修改将游戏的数值系统进行了全面重构，提升了等级上限，新增命中/闪避/暴击三大战斗属性，并调整了所有属性的上限值。

## 修改内容清单

### 1. 等级上限：30 → 99
- 默认经验上限扩展到 999999
- 经验表（levelup.txt）会自动扩展到99级，每级默认60000经验

### 2. 新增三大战斗属性
| 属性 | 说明 | 默认上限 |
|-----|------|---------|
| HitRate | 命中率 | 100 |
| Dodge | 闪避率 | 100 |
| CriticalRate | 暴击率 | 100 |

### 3. 属性上限调整
| 属性 | 旧上限 | 新上限 | 说明 |
|-----|--------|--------|------|
| HP/MaxHP | 999 | **9999** | 配合99级增加的血量成长 |
| MP/MaxMP | 999 | **9999** | 同上 |
| Attack | 100 | **999** | 高手攻击力应远超常人 |
| Defence | 100 | **999** | 配合攻击力 |
| Speed | 100 | **200** | 轻功差异大但不应过分 |
| Medicine | 100 | **200** | 医疗能力 |
| UsePoison | 100 | **200** | 用毒能力 |
| Detoxification | 100 | **200** | 解毒能力 |
| AntiPoison | 100 | **200** | 抗毒能力 |
| Fist/Sword/Knife/Unusual/HiddenWeapon | 100 | **200** | 武功属性 |
| Knowledge | 100 | **200** | 武学常识 |
| AttackWithPoison | 100 | **200** | 攻击带毒 |

### 4. 战斗公式调整

#### 命中判定（新增）
```
实际命中率 = min(95, max(5, 攻击方.HitRate - 防御方.Dodge + 50))
```
- 基础命中率为50%
- 命中率在5%~95%之间浮动
- 命中判定失败显示 "MISS"

#### 暴击判定（修改）
- **旧逻辑**：硬编码20%概率
- **新逻辑**：使用 `攻击方.CriticalRate` 属性
- 暴击伤害：1.5倍

### 5. 升级成长调整
每次升级：
- HitRate: +0~2
- Dodge: +0~1
- CriticalRate: +0~1

## 修改的文件

### 核心数据结构
- `src/Types.h` - RoleSave 结构体添加新属性字段
- `src/NewSave.h` - 添加新属性的序列化定义

### 属性限制与上限
- `src/Types.cpp`
  - `setMaxValue()` - 修改所有属性上限，添加新属性上限读取
  - `limit()` - 添加新属性限制
  - `levelUp()` - 添加新属性增长

### 战斗逻辑
- `src/BattleScene.cpp`
  - `calMagiclHurtAllEnemies()` - 添加命中判定，修改暴击逻辑
  - `actUseHiddenWeapon()` - 暗器攻击添加命中判定和暴击

### UI显示
- `src/ShowRoleDifference.cpp` - 添加新属性的升级前后对比显示
- `src/UIStatus.cpp` - 添加新属性在角色状态界面显示

## 配置文件
属性上限可通过 `game/kysmod.ini` 的 `[constant]` 节自定义：

```ini
[constant]
Level = 99
HP = 9999
MP = 9999
Attack = 999
Defence = 999
Speed = 200
HitRate = 100
Dodge = 100
CriticalRate = 100
; ... 其他属性
```

## 存档兼容性
由于新增了三个属性字段，旧存档加载时这些属性会使用默认值0。
建议用户开始新游戏以体验完整的数值系统。
